# 天宇物业管理系统

基于Spring Boot和Vue 3的现代化物业管理解决方案

## 项目特点

- 多项目数据隔离
- 多物业适配
- 动态收费规则
- 智能支付系统
- 分级权限控制
- 数据安全保障

## 技术栈

### 后端

- Spring Boot 2.7.13
- Spring Security
- MyBatis-Plus 3.5.3
- MySQL 8.0
- Redis

### 前端

- Vue 3
- Element Plus
- Pinia
- Vue Router
- Axios

## 开发计划

### 第一阶段：基础架构完善（2周）
1. 实现数据隔离机制
2. 搭建RBAC权限框架
3. 完善数据库设计

### 第二阶段：核心功能开发（3周）
